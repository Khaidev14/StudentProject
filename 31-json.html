JAVASCRIPT OBJECT NOTATION

What is JSON in JavaScript?

   JavaScript Object Notation, or JSON, is a simple data exchange format that is simple for computers to understand and produce as well as for people to read and write. JSON is language-independent, meaning it may be used in many computer languages, while being built on a subset of JavaScript.
