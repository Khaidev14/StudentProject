FOR LOOP

What is For Loop?

   A for loop in JavaScript is another type of loop that allows you to execute a block of code repeatedly. It's particularly useful when you know in advance how many times you want to execute the loop.

Three parts of For Loop:

1. Initialization: Initializing the variable to be used before the loop starts.

2. Condtion: Defining and setting the condition that must be true depending on how many times you want the loop to be executed.

3. Incrementing/Decrementing: Updating the value of the variable every iteration of the loop

What is the difference between while loop and for loop?

   while loops are more flexible and suitable for scenarios where the number of iterations is not predetermined, while for loops are more concise and convenient for iterating over a fixed range of values or elements. Both loops have their advantages and are used based on the specific requirements of the task at hand.

Additional information:

   By default, for loops iterate in ascending order, starting from the initial value and ending at the condition. For example, for (let i = 0; i < 5; i++) will iterate from 0 to 4. However, you can customize the loop to iterate in descending order by adjusting the initialization, condition, and increment/decrement. For instance, for (let i = 5; i > 0; i--) will iterate from 5 down to 1.

   Remember that while for loops are powerful and widely used, it's essential to be mindful of loop conditions, initialization, and updates to avoid potential pitfalls like infinite loops or unexpected behavior.
