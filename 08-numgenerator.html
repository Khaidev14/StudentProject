RANDOM NUMBER GENERATOR

Random Number Generator is a project designed to randomly generate numbers within a designated range. In this project's case, the minimum number to be generated is 1 and the maximum is 100. This will generate numbers starting from 1 down to 100.

Functionality: The application uses JavaScript's built-in `Math.random()` to generate specific numbers within a specific range. This function can generate floating-point numbers between 0 to 1. By using this function and multiplying it to the maximum number as a range and adding into the equation, we can generate random numbers.
