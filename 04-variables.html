What is Variable?

   In JavaScript, a variable is a storage or container that contains a specific whether be it an integer, string, characters or even dates. It is like a storage with a name that you can manipulate and change over a specific course of time. It is one of the most essential parts of programming as it is used to manipulate data inside your system.

   In JavaScript, there are three(3) keywords you can use to store a value and create a variable. We can create variable using `var`, `let` and `const`. But, the keyword `var` is considered to be outdated and may cause problems that you may encounter as you progress coding your own website. Therefore, WOJA will only be using `let` and `const` to avoid unecessary errors for beginners.
