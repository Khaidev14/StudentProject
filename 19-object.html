What is Object in JavaScript?

   A collection of key-value pairs, where each key is a string (or symbol) and each value can be any data type, including other objects, is represented by an object in JavaScript as a composite data type. One of the core ideas of JavaScript is the object, which is used to represent intricate entities and data structures.

Basic Syntax of Object:

In this particular example, person is the name of the object variable. The object is enclosed in curly braces {}. Inside the object, there are three key-value pairs:

"name" is a key, and "John" is its corresponding value.
"age" is a key, and 30 is its corresponding value.
"city" is a key, and "New York" is its corresponding value.

   JavaScript objects are mutable, which means that once an object is formed, its properties can be updated or changed. An object has dynamically added new properties, updated existing properties, and deleted properties.

   JavaScript objects are extensively used and adaptable; they can be used to represent real-world entities, organize data, and create more intricate data structures and applications. JavaScript developers must comprehend objects and how to manipulate them.
