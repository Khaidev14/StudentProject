THIS KEYWORD in METHOD CONTEXT

What is This Keyword in JavaScript?

   This is a JavaScript keyword that describes the context in which a function is run. This might mean multiple things depending on the context and is dependent on how a function is invoked.

   In method context, `this` keyword refers to the object that owns the method.

For example:

In this particular example:

· The brand, model, and year of the car are among its properties.
· Additionally, it provides a function called getInfo that produces a string with details about the make, model, and year of the vehicle.
· This is the actual car object, as used in the getInfo method. The properties of the car object are thus accessed by this.brand, this.model, and this.year when getInfo is called on the car object using car.getInfo().

   In summary, When a method is included within an object, the method's owning object is referred to by the this keyword. This behavior increases the flexibility and reusability of programming by enabling methods to dynamically access and modify the object's properties.
