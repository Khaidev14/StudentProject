ELEMENT SELECTOR

What is DOM navigation?

   The technique of using JavaScript to go around a web page's Document Object Model (DOM) tree is known as "DOM navigation." A web page's structure is represented by the Document Object Model (DOM) as a hierarchical tree of objects, with each node denoting an element, attribute, or text paragraph.

   Developers may programmatically access and modify many web page elements with DOM navigation. This can involve activities like locating certain elements, changing their characteristics or content, adding or deleting elements, and reacting to user input.

There are multiple ways how to perform DOM navigation, we can use `Element Selectors` to perform DOM Navigation.

What is Element Selector in JavaScript?

    JavaScript element selectors are ways to use JavaScript code to target and select particular HTML components within the Document Object Model (DOM) of a web page. Web applications that are interactive and responsive are made possible by these selectors, which give developers the ability to dynamically access and modify items.

Here are some examples of Element Selector in JavaScript:

1. getElementById(): This method selects an element by their unique identification or ID.

2. getElementsByClassName(): This method particularly selects an element based on their class names or attributes.

3. getElementsByTagName(): This methods selects an element based on their tag names inside your HTML file.

4. querySelector(): This methods selects the first element of matching class names or ID.

5. querySelectorAll(): This method selects all the element and returns a nodelist containing all the selected element.

   These selectors give programmers strong capabilities for gaining access to and modifying web page elements. They are frequently used in JavaScript programming to let online applications to dynamically change their behavior, style, and content in response to user input and other events.
