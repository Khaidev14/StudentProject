TEMPERATURE CONVERSION

Temperature Conversion

Temperature Conversion: This program is a temperature conversion. It can perform celsius to fahrenheit conversion and vice versa. There is no such amount of limit with this program and you can convert large amount of numbers.

BRIEF TUTORIAL AND GUIDE

   We can start by creating our HTML document file and creating necessary components, elements and containers for our project. You can follow these steps and include it in your HTML file and style it using CSS. If you want to view the CSS code for this project, you can visit the source code at the end of the page.

   After creating your html file and styling your project using css, we can now move on and create functionalities of some elements in our file. Let us jump towards our JavaScript file and select the submit button, two(2) radio buttons and display result using element selector. After selecting necessary elements, create a function whenever we click the `submit button`.

After creating the function we need to start the program, we have to verify some information to proceed. Add and `if` statement inside your function to verify whether the input box is not null or has value and select the input box using element selector and store it in a constant variable. Add an `else` statement after your `if` statement.

Then, create an `if`, `else if` and `else` statement inside your main `else` statement. Use an `if` and `else if` statement to verify whether the conversion method is celsius to fahrenheit or vice-versa. After evaluating which method will be used, we can now proceed to the development of the computation to get the right result using the default formula of computing celsius - fahrenheit and vice-versa and display it.
