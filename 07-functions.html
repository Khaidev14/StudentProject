Purpose of Function in JavaScript

Code Simplicity and Organization: Functions allow you to create and break down your code into a concise units which is reusable. Functions usually intends to be included especially to reduce code repetitions.

Code Reusability: Once declared and called, you can call your declared functions multiple times within the right scopes. It will prevent redundancy and promote reusability.

Return Values: Function is capable of returning specific values which allows you to perform an order computations to produce results. These results can be returned and is reusable.

Event Handling: In JavaScript and Web Development, functions are usually use to handle different events triggered by the user. This events include mouse over, hovering, clicking, keyboard pressing and more.

   Overall, functions are used often and is essential building block of JavaScript programming. This will enable you to write reusable codes, maintainable and impressive codes that can handle tasks and computations.
