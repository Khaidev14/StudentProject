What is Switch Case in JavaScript?

   In JavaScript, switch statement is often known as controller of flow statement that allows you to execute a specific block of code based on the value of an expression given by a specific condition. It is also often used as an alternate to `if` and `else` statement especially if it is a long chain structure.

Here is how it works:

The switch expression is evaluated once.
1. The value of the expression is compared with the values of each case.
2. If a match is found, the code block associated with that case is executed.
3. If no match is found, the code block associated with the default case (if present) is executed.
4. The break statement terminates the switch statement. Without break, execution falls through to the next case.

   Switch cases is often and particularly being used when you have only one expression to be evaluated against many possible outcomes. Switch case statement is often used such as dealing with menu options, user input, or different parts of programs that requires decision making.
