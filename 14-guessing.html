RANDOM NUMBER GENERATOR

Instruction: Generate a secret number and guess it!

Random Number Generator: Random number generator is a simple program created that will prompt the user to generate the a random number and guess it with the help of small clues given by the simple project. The number range in Random Number Generator is 1 to 50. The minimun value and the maximum value has already been set by default.

Purpose: This is to display and utilize our skills using the previous lessons we conducted such as if and else statements, element selectors and variable storing.

Note: To further Enhance the simplicity of this lesson, I prepared a project we can discuss similar to this one. In this project, we will be discussing a much more basic number guessing game where we can prompt the user to input a number value using window prompt and displaying the result in your console.

BRIEF TUTORIAL AND GUIDE

In this tutorial, we will be covering some of the lessons we previously discussed. Here is the overall interface of our project in this course:

Let's start by discussing the first step!

First, create your HTML file and its basic structure.

Next, declare all necessary variables you need such as the minimun number, maximum number and the number of attempts to that the user committed. After declaring, it is now time to declare the variable `answer` which will hold the secret number to be guessed.

After following the codes given and declaring all the variables needed, we can now proceed to the creation of while loop that will determine whether the program shall continue or stop depending on a specific situation. Then, prompt the user to enter their guess and store it in `guess` variable - declare the `guess` variable as number.

After that, inside your while loop, add an `if` statement that will check if the `guess` variable is not a number. If the guess variable isn't a number, we will be prompting the user again by saying "Please enter a valid number". Add this code inside your while loop.

After inserting adding an `if` statement, add an else if statement below it to check if the guess of the user is outside the range of minimun and maximum variable.

lastly, perform nested if and else statement by adding if and else inside your else statement that will check whether the guess is too high, too low or correct.
