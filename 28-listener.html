EVENT LISTENER

What is Event listener in JavaScript?

   In JavaScript, an event listener is a function that watches for a particular event, like a click, mouseover, keypress, or page load, to happen on an HTML element and then triggers a predefined piece of code. A key component of creating dynamic, interactive websites and web apps is event listeners.

There are several event type we can use such as:

1. click: Executed when the user clicked on an element.

2. mouseover: Executed when the user hover over an element.

3. mouseout: Executed when the user hover out on an element.

There are also keyboard events which can be triggered when user pressed down a specific keys on keyboard. For example:

1. keydown: Executed when the user pressed down a key.

2. keyup: Executed when pressed key has been released.

3. keypress: Executed when the user presses and releases a key.

   These are just a few of the numerous JavaScript event kinds that are accessible. Developers are able to design rich and engaging web apps since each event type is associated with a particular action or interaction that might take place within the browser environment.
